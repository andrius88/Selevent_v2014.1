
Program SELEVENT reads  NORSAR GBF bulletins (html format) and 
selects events confined by some rectangular area. The output 
file (GBF??_sei.txt) contains events in NORDIC format (compact 
or S-files).

NORSAR GBF files (html files) of one year (or part of the year)
 should be stored in one directory.
From menu -> file -> open should be open the first file of that
 directory. Program will read all files (day by day) to the end
 of the month/year.
User should choose compact or S-files should be given in output.
User should indicate the co-ordinates of rectangular area from which
 events should be selected.
