
Programele Selevent "ima" Norsar GBF biuletenius html formate ir
pagal uzduotas koordinates isrenka ten uzfiksuotus ivykius. Juos galima 
pasiimti is interneto. 

meniu -> file -> open reikia atidaryti pirma
faila nuo kurio bus pradetas ivykiu isrinkimas. Programele skaitys 
visus failus iki tu metu pabaigos, todel reikia, kad vienu metu failai 
butu atskirije direktorijoje. Tada nustatyti norimas koordinates, 
pasinkti iseinancio failo tipa (S-file, compacat) ir pasusti 
"select+convert", kad paleisti procesa.
